# blog

My personal blog

## Como executar o projeto

1. Instale as dependências:

   ```bash
   npm install
   ```

2. Execute o servidor local:

     ```bash
    npm start
     ```

3. Execute os testes:

     ```bash
    npm test
     ```

## Tools Used in This Project

- [GitHub Copilot](https://github.com/features/copilot)
- [Mocha](https://mochajs.org/)
- [Jest](https://jestjs.io/)
